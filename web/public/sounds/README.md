# Soundscape assets

Place royalty-free mp3 loops here, named to match `src/features/audio/manifest.ts`:

- forest.mp3   — distant birds + leaves
- ocean.mp3    — slow rolling surf
- rain.mp3     — steady soft rainfall
- fire.mp3     — crackling embers
- wind.mp3     — high mountain wind
- waves.mp3    — gentle lapping water

**Suggested sources** (CC0 / public domain — verify license before commit):
- https://pixabay.com/sound-effects/search/ambient/
- https://freesound.org/ (filter CC0)

**Targets:** 1–3 minute loops, stereo, ~96–128 kbps, < 2 MB each.
The player loops them seamlessly via Howler with html5: false.
